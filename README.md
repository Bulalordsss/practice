# Practice App

## How to run the app

1. Install dependencies

```bash
npm install
```

2. Start the project

```bash
npx expo start
```

3. Open it using Expo Go, Android emulator, or iOS simulator

## Tech used

- Expo
- React Native
- Expo Router
- Axios
- Zustand
- Expo SQLite
- TypeScript

## Features implemented

- Login screen with route protection
- Home screen with welcome header and summary cards
- Recent activities fetched from API
- Pull to refresh on home screen
- Local To-Do creation using SQLite
- To-Do list screen with search and filtering
- Edit task title, details, and status
- Soft delete or hide task instead of permanent delete
